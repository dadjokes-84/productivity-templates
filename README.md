## 🚀 How to Use

### For Notion Templates
1. Open Notion and create a new page
2. Open the `.md` file in a text editor
3. Copy all content and paste it into your new Notion page
4. Notion will automatically render the markdown tables and headings
5. Duplicate the page each time you want a fresh copy (e.g., each week/month/project)

### For Obsidian Templates
1. Copy the `.md` file into your Obsidian vault's templates folder (e.g., `_templates/`)
2. Open Obsidian Settings → Core Plugins → enable **Templates** (or install **Templater** for the `{{date}}` variables)
3. Set your templates folder in Settings → Templates → Template folder location
4. Create a new note and use Ctrl+P → "Insert template" to apply it

> **Templater Plugin (recommended for Obsidian):** The `{{date:...}}` syntax in the daily notes template works automatically with the Templater community plugin. Install it from Obsidian's Community Plugins panel.
